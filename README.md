# 🧠 AI-Powered Talent Scouting & Engagement Agent

An intelligent recruitment tool that automates candidate sourcing and engagement. Given a job description, it parses key requirements, matches them to candidate profiles using NLP-based similarity scoring, and simulates conversational outreach to gauge interest. Outputs a ranked list of candidates based on **Match Score** and **Interest Score**, helping recruiters prioritize outreach efficiently.

Built with Flask, scikit-learn, and rule-based chat simulation — perfect for rapid prototyping in recruitment tech.

---

## ✨ Features

- Upload CSV of candidate profiles
- Match candidates using TF-IDF + cosine similarity
- Simulate engagement/conversational interest
- Rank candidates by combined Match + Interest Score
- Easy-to-use web interface
